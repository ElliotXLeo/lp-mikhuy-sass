Landing Page de Restaurante Chanchirata donde se cuenta con un Hero que expone la información más importante de la web. Desarrollado con Sass.

## 💻 Instalación 💻
- Ubicarse en la carpeta que contendrá el proyecto
- Abrir terminal de comandos
  - git clone https://github.com/ElliotXLeo/lp-mikhuy-sass
  - cd lp-mikhuy-sass
  - Levantar la web en un servidor y disfrutar
